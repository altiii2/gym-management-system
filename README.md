# Gym Management System - OOP Project

## Week 2: Classes, Objects, Methods, and Constructors

### Project Overview
This is a Gym Management System built with Java OOP principles. It demonstrates:
- Class creation with private fields
- Constructors (parameterized and default)
- Getters and setters
- Additional methods with business logic
- Proper encapsulation

### Classes Implemented
1. **Member** - Represents a gym member
2. **Trainer** - Represents a fitness trainer
3. **WorkoutSession** - Represents a training session

### Requirements Met
- ✅ 3+ classes (not counting Main)
- ✅ 4+ private fields per class
- ✅ Constructors with parameters
- ✅ Getters and setters for all fields
- ✅ 2+ additional methods per class
- ✅ toString() methods
- ✅ Main class with comprehensive demonstration
